專案紀錄與 PWA 建立教學

📌 初心與目標

我想整一個工具集 Dashboard 式網站，目標係將一啲日常會用到嘅小工具集中放喺同一個地方，之後可以慢慢再加新工具。

第一版先放 3 個工具：縮短網址工具、番茄時鐘、QR Code 產生器。
我想要嘅唔係 3 個分開嘅工具頁，而係一個有整體感、似真正產品嘅小工具 Dashboard。
首頁可以睇到工具卡片，按入去就可以使用各自功能。整體風格走簡約、有設計感、大地色系。

請幫我規劃頁面結構、設計首頁同工具頁、建立第一版可運作 MVP，令 3 個工具都真係可以用到，並支援 PWA（可加到手機主畫面）。

3 個工具功能要求：
• 縮短網址：輸入長網址生成短連結，可複製。
• 番茄時鐘：25 分鐘專注 + 5 分鐘休息，有開始、暫停、重設功能。
• QR Code 產生器：輸入文字或網址即時生成 QR Code，可下載圖片。

💡 AI 開發與設計思路

呢個 Dashboard 嘅概念非常好！將日常工具集中喺一個有設計感嘅平台，唔單止自己用得舒服，日後擴展做 SaaS 或者開源項目都好有潛力。

為咗配合你要求嘅「簡約、大地色系 (Earth Tones)、有整體感」，我哋會用米白色作為背景，配搭溫暖嘅棕色同灰褐色作為主色調。為咗令你可以最快測試呢個 MVP（最小可行產品），我會將 HTML、CSS 同 JavaScript 寫喺同一個 index.html 檔案入面，並透過簡單嘅 DOM 切換嚟模擬單頁應用（SPA）嘅流暢體驗。另外會提供 PWA 必需嘅 manifest.json 同 Service Worker。

📁 專案檔案結構

喺電腦開一個新資料夾（例如 tool-dashboard），包含以下 3 個核心檔案：

  • 1. index.html (主程式、UI 同邏輯)： 包含咗大地色系嘅 CSS 設計、Dashboard 卡片佈局，以及 3 個工具嘅完整運作邏輯。
  • 2. manifest.json (PWA 設定)： 呢個檔案會話畀瀏覽器知，呢個網站可以安裝成為手機 App。
  • 3. sw.js (Service Worker)： 加入 Service Worker 可以令你嘅網站喺手機加落主畫面後，即使網絡唔穩定都可以秒速開啟 UI，支援離線快取。

🚀 PWA 部署與上線步驟

既然已經準備好上述 3 個檔案，接下來嘅核心目標係測試同埋發佈：

步驟一：喺自己電腦做本地測試 (Local Test)

  1. 基於瀏覽器安全限制，PWA 嘅離線快取技術 (Service Worker) 唔允許你直接 double-click 檔案（即 file:/// 網址）嚟運作。你需要一個本地伺服器。
  2. 使用 VS Code 編輯器，去左側 Extension 搜尋並安裝「Live Server」。
  3. Right-click index.html，選擇「Open with Live Server」。
  4. 瀏覽器會彈出類似 http://127.0.0.1:5500 嘅網址，你就可以喺電腦完美測試。

步驟二：建立 GitHub Repository (儲存庫)

  1. 登入 GitHub 後，喺右上角撳 + 號，揀「New repository」。
  2. 設定名稱 (例如 toolbox-dashboard) 並設為 Public，撳「Create repository」。
  3. 點擊「uploading an existing file」，將 3 個檔案直接拖曳上傳。
  4. *(注意：確保檔案直接放喺最外層，唔好包喺資料夾入面)*。
  5. 拉到底部點擊「Commit changes」儲存。

步驟三：喺 Vercel 免費部署上網

  1. 登入 Vercel 點擊「Add New...」>「Project」。
  2. 選擇從 GitHub Import，並授權 Vercel 讀取你嘅帳戶。
  3. 搵返頭先建立嘅 toolbox-dashboard，撳「Import」並直接「Deploy」。
  4. 等待幾十秒，Vercel 就會提供一條真實網址。日後你喺電腦改完 Code 上傳 GitHub，Vercel 就會自動更新網站。

步驟四：手機 PWA 實測 (Mobile Test)

  1. 用手機瀏覽器打開 Vercel 提供嘅網址。
  2. iPhone 用戶 (Safari)： 撳畫面底部的「分享」掣，慢慢向下拉，揀「加至主畫面 (Add to Home Screen)」。
  3. Android 用戶 (Chrome)： 瀏覽器通常會主動彈出提示；如果冇，可以撳右上角三點選單揀安裝。
  4. 裝完之後返回手機桌面打開，你會發現冇咗瀏覽器嘅網址列，好似一個原生 App 咁！
